# AI Playbook

An **evolving playbook** of durable lessons for AI assistants working in this repo.
Inspired by ACE (Agentic Context Engineering): accumulate and refine strategies over
time as small, additive deltas — never rewrite or summarize away prior lessons, which
causes "context collapse."

How to use it:
- **Read** this file before starting non-trivial work (it is cheap, persistent context).
- **Append** a lesson when you discover something durable: a gotcha, a convention, a
  command that works, a dead end to avoid. Use `aito learn "<lesson>"` or add a bullet
  under Lessons. Keep each lesson one line, specific, and testable.

## Conventions

- (seed) Prefer existing repository patterns over introducing new ones.
- (seed) Keep secrets out of any context, prompt, or generated artifact.
